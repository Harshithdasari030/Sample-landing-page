# Sample-landing-page
# 🚀 Responsive Landing Page

A modern and responsive landing page built using **HTML5 and CSS3**.  
This project demonstrates a clean UI design with sections like hero banner, features, call-to-action, and footer.

It is suitable for **product websites, startup landing pages, or portfolio projects**.

---

## 📌 Project Overview

This landing page provides a simple yet attractive layout that helps businesses showcase their product or service.  
It includes a responsive navigation bar, hero section, feature highlights, and a call-to-action section designed to improve user engagement.

The design is **mobile-friendly, lightweight, and easy to customize**.

---

## ✨ Features

- 📱 **Responsive Navigation Bar**
- 🎯 **Hero Section with Call-to-Action Buttons**
- ⚡ **Features Section with Cards**
- 📊 **Statistics Section**
- 🔗 **Call-To-Action (CTA) Section**
- 📄 **Footer with Links and Social Icons**
- 🎨 **Modern UI using Google Fonts**
- 💻 **Mobile Responsive Layout**

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the web page  
- **CSS3** – Styling and layout design  
- **Google Fonts (Inter)** – Modern typography
